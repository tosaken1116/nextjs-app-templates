## Next.js AppRouter用のテンプレート

### 使用手順
1. clone

        git clone git@github.com:tosaken1116/nextjs-app-templates.git

    or

        git clone https://github.com/tosaken1116/nextjs-app-templates.git

2. ディレクトリ移動

        cd nextjs-app-templates

3. パッケージマネージャの選択

        chmod +x setup.sh

        ./setup.sh
